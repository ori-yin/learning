# 📚 学习笔记

个人学习资料汇总，包含 Python、机器学习、深度学习等内容。

## 目录结构

```
learning/
├── python/                    # Python 基础与工具
│   ├── pyechart可视化/        # Pyecharts 数据可视化
│   ├── python连接SQL/         # Python 数据库操作
│   ├── 小工具/                # 实用小工具脚本
│   └── 日常工作/              # 工作中常用的 Python 脚本
├── machine-learning/          # 机器学习
│   ├── sklearn入门/           # scikit-learn 学习笔记 (原 sklearn-learn)
│   ├── hands-on-ml/           # 《Hands-On ML》Jupyter 笔记 (原 Machine-learn)
│   └── 实战练习/              # 机器学习实战代码
├── deep-learning/             # 深度学习
│   ├── tensorflow/            # TensorFlow 相关
│   └── keras/                 # Keras 相关
├── ai/                        # AI 综合
│   └── 环境配置/              # AI 开发环境搭建指南
└── README.md                  # 本文件
```

## 来源说明

本项目由以下仓库合并而成：

| 原仓库 | 新位置 | 说明 |
|--------|--------|------|
| `Python` | `python/` | Python 基础学习 |
| `AI_learning` | `ai/` | AI 学习资料 |
| `sklearn-learn` | `machine-learning/sklearn入门/` | scikit-learn 课程笔记 |
| `Machine-learn` | `machine-learning/hands-on-ml/` | 《Hands-On ML》笔记 |

```bash

```
